解决方案说明

通过CSS媒体查询，我们可以针对不同的打印方向设置不同的样式：

  • 使用 @page 规则设置页面方向
  • 使用 @media (orientation: portrait) 针对纵向打印进行优化
  • 在纵向打印时调整字体大小、边距和表格布局
  • 使用 word-break: break-word 确保长文本不会溢出
